---------------------------------------------------
4. AZURE VM REQUIREMENTS
---------------------------------------------------
- OS: Ubuntu Server 22.04 LTS
- Size: Standard_B2s (2 vCPUs, 4 GB RAM) minimum
- Disk: 30 GB Standard SSD
- Ports: 22 (SSH), 80 (HTTP), 443 (HTTPS)
- Authentication: SSH public key (use the PUBLIC KEY above)

---------------------------------------------------
5. QUICK SSH CONNECTION
---------------------------------------------------
After VM is created, connect with:

ssh -i ~/.ssh/azure_vm_key azureuser@YOUR_VM_PUBLIC_IP

Example:
ssh -i ~/.ssh/azure_vm_key azureuser@20.123.45.67
